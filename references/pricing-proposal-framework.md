# 法律服务报价方案框架

> 共享参考文件——legal-clinic 和洽商/报价相关技能使用本框架。
> 覆盖从客户需求分析到最终方案输出的全流程。

---

## 流程总览

```
Phase 1  项目启动与意图分析
Phase 2  客户需求结构化分析
Phase 3  知识库检索与研究
Phase 4  服务策略制定
Phase 5  报价策略选择与计算
Phase 6  方案撰写
Phase 7  质量审核
Phase 8  方案输出
```

---

## Phase 1：项目启动与意图分析

**输入**：项目背景信息、具体法律需求说明、报价策略方向。

**项目分类**——判断属于三类方案中的哪一类：

| 分类信号 | 方案类型 |
|----------|----------|
| 起诉、应诉、仲裁、争议、纠纷、诉讼代理、执行 | 诉讼/仲裁代理及报价方案 |
| 尽调、并购、重组、融资、合规审查、专项、项目 | 非诉讼专项法律服务报价方案 |
| 常年、顾问、年度、日常法律服务 | 常年法律顾问服务报价方案 |

**输出**：项目文件夹结构 + 项目摘要（含分类结论）。

---

## Phase 2：客户需求结构化分析

### 分析维度

| 维度 | 分析内容 |
|------|----------|
| 客户画像 | 行业、规模、发展阶段、关键决策人 |
| 法律需求 | 核心法律问题、涉及法律领域、复杂度评估 |
| 痛点与期望 | 客户最关心什么、过往痛点、期望的服务方式 |
| 预算敏感度 | 对价格的敏感程度、决策流程、历史付费水平 |
| 关系深度 | 新客户/老客户/战略客户、未来业务潜力 |
| 竞争态势 | 是否有竞争对手报价、客户比较标准 |
| 时间约束 | 方案提交截止日期、服务启动时间 |

### 复杂度评估

| 级别 | 特征 | 对报价的影响 |
|------|------|-------------|
| 高 | 涉及多法律领域、金额大、时间紧、事实复杂 | 高费率、充足工时、资深团队配置 |
| 中 | 单一法律领域但有一定复杂性 | 标准费率、合理工时 |
| 低 | 标准化需求、事实清楚、法律关系简单 | 可考虑固定收费或标准套餐 |

---

## Phase 3：知识库检索与研究

1. **同类服务检索**：查找同类法律服务产品（服务内容设计、方法论、模块划分、服务承诺标准）
2. **法律问题研究**：项目涉及的核心法律问题、适用法规、主要风险点、典型处理流程
3. **法律障碍与服务可行性评估**：识别哪些法律困难影响定价、哪些构成承接障碍（如交易路径不确定、监管新规影响、决策依据效力瑕疵）
4. **行业价格参考**：同类服务的定价方式和市场价格区间

---

## Phase 4：服务策略制定

### 服务范围界定

| 类别 | 定义 | 报价处理 |
|------|------|----------|
| 核心服务 | 客户明确要求的、项目必须包含的服务 | 包含在基础报价中 |
| 推荐服务 | 客户未明确要求但基于专业判断建议增加的 | 单独列明，标注推荐理由 |
| 可选增值 | 提升服务价值的附加服务 | 单独报价，供客户选择 |
| 排除事项 | 明确不包含的服务 | 在方案中明确列出，避免误解 |

### 团队配置

| 角色 | 职责 | 适用场景 |
|------|------|----------|
| 主办律师 | 项目统筹、核心法律判断、客户沟通 | 所有项目 |
| 协办律师 | 具体法律研究、文书起草、证据整理 | 中高复杂度 |
| 律师助理 | 资料整理、基础检索、行政支持 | 按需 |
| 专业顾问 | 特定领域专家（税务、评估等） | 涉及专业领域时 |

## Phase 5：报价策略选择与计算

### 报价方式决策矩阵

| 报价方式 | 适用条件 | 优势 | 风险 |
|----------|----------|------|------|
| **固定收费** | 服务范围明确、工作量可预估 | 客户接受度高、预算可控 | 范围变更需重新议价 |
| **计时收费** | 工作量不确定、复杂度高 | 灵活、公平 | 客户预算焦虑 |
| **风险代理** | 诉讼/仲裁、有明确经济诉求、胜诉概率较高 | 客户门槛低、利益绑定 | 收入不确定、合规限制 |
| **固定+计时混合** | 部分工作可预估、部分不确定 | 平衡确定性 | 结构复杂需解释 |
| **固定+风险混合** | 诉讼案件，基础工作+结果奖励 | 兼顾保障和激励 | 需合理设定比例 |
| **分阶段收费** | 项目周期长、有明确阶段节点 | 风险分散、现金流好 | 需明确定义各阶段 |

### 选择逻辑

```
项目类型 → 核心判断 → 报价方式
诉讼/仲裁 → 标的额明确 + 胜诉概率评估 → 风险代理 或 固定+风险混合
诉讼/仲裁 → 标的额不确定或复杂 → 固定 或 固定+计时
非诉专项 → 范围明确 → 固定
非诉专项 → 范围不确定 → 分阶段 或 固定+计时
常法顾问 → → 固定年费 + 超出范围计时
```

### 费用调整机制

当服务范围取决于未来不确定因素时（交易路径未确定、法律政策环境变化、第三方行为不确定、服务范围可能扩展），必须在报价方案中设计费用调整机制：明确定义触发情形 → 说明增加工作量的原因 → 给出调整幅度区间 → 设计双方确认程序。

### 报价合理性自检

| 检查项 | 标准 |
|--------|------|
| 工时合理性 | 各角色工时预估是否与工作量匹配 |
| 费率合规性 | 是否符合当地收费标准和律所定价政策 |
| 利润率 | 项目利润率是否在可接受范围 |
| 市场竞争力 | 与同类服务市场价格相比是否合理 |
| 内部一致性 | 总价 = 各模块明细之和 |
| 谈判弹性 | 是否预留了谈判空间 |

## Phase 6-8：方案撰写 → 质量审核 → 输出

### 三类方案结构

**诉讼/仲裁代理及报价方案**：
```
封面 → 案件概况与初步分析 → 代理策略 → 团队配置 →
服务内容与报价 → 服务承诺 → 合作条款 → 附件
```

**非诉讼专项法律服务报价方案**：
```
封面 → 项目理解 → 服务内容与范围 → 服务方法论 →
团队配置 → 报价方案 → 服务承诺 → 合作条款 → 附件
```

**常年法律顾问服务报价方案**：
```
封面 → 服务理念 → 服务内容（基础+增值） → 团队配置 →
年度报价 → 服务承诺 → 合作条款 → 附件
```

### 撰写原则

- **面向客户**：方案是给客户看的，不是内部文件
- **突出价值**：先说能为客户解决什么问题，再说怎么做
- **专业克制**：不过度承诺，不贬低同行
- **清晰明确**：服务范围、报价、条件无歧义

### 质量审核清单

| 类别 | 检查项 |
|------|--------|
| **完整性** | 方案是否包含所有必要章节；客户需求是否全部回应 |
| **一致性** | 报价金额是否全文一致；服务范围描述是否前后统一 |
| **准确性** | 法律分析是否正确；引用的法规和案例是否准确 |
| **竞争力** | 差异化是否突出；与竞争对手相比优势在哪 |
| **合规性** | 报价是否符合当地收费管理规定；风险代理比例是否合规 |
| **客户视角** | 客户能看懂吗；会不会产生误解 |

### 利益冲突审查

项目启动阶段同步进行：对方当事人是否与本所现有客户存在利益冲突；是否曾代理过对方或类似案件；是否涉及前客户保密信息。发现潜在冲突，立即提示用户，不继续推进。
